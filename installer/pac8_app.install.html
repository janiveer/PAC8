Installer

This menu enables an installer to record the outcome of a site visit.

  1. Choose an option:

    Installation Complete

    Logs the installation as complete.

    Repair Complete

    Logs the repair as complete.

    Second Visit Required

    Logs the site as requiring a second visit.

  2. When you’re ready, tap ← to return to the Main menu.
