Keys

This menu displays a list of configured keys. The top line shows the number of configured keys.

To configure an existing key:

  1. Tap the name of the key.

    The Key Details menu is displayed.

  2. When you’re ready, tap ← to return to the Main menu.

To add a new key:

  1. Tap Add.

    The Key Details menu is displayed.

  2. When you’re ready, tap ← to return to the Main menu.
