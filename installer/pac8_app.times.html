Time Profiles

This menu displays a list of configured time profiles. The top line shows the number of configured time profiles.

To configure an existing time profile:

  1. Tap the name of the time profile.

    The Time Profile Details menu is displayed.

  2. When you’re ready, tap ← to return to the Main menu.

To add a new time profile:

  1. Tap Add.

    The Time Profile Details menu is displayed.

  2. When you’re ready, tap ← to return to the Main menu.
