Events

This menu enables you to view recent events from the connected PAC8 controller.

To view events:

  1. To download events from the controller, tap Sync.

    A list of recent events is displayed. This shows all the events that the controller has logged since the last time you downloaded events.

  2. To view details of an event, tap the event.

    The Event Details page is displayed.

  3. When you’re ready, tap ← to return to the Synchronisation menu.

Note

The list of downloaded events is cleared when you leave this menu, or if you tap Sync again.
