Firmware Update

This menu enables you to view the current controller firmware and update it if necessary.

To check the controller firmware:

  1. Check the following options:

    Current Firmware

    The version number of the current PAC8 controller firmware.

    Available Firmware

    A list of available firmware images.

  2. To update the controller firmware, tap the firmware image you want.

    The firmware is downloaded and installed. The status of the process is displayed at the bottom of the screen.

  3. When you’re ready, tap ← to return to the Synchronisation menu.
