Synchronisation

This menu enables you to work with a connected PAC8 controller via Bluetooth.

To work with a connected controller:

  1. Choose an option:

    Synchronise Controller

    Uploads settings and key data from your mobile device to the controller. Existing settings and key data on the controller are overwritten.

    When the warning message appears, tap OK to continue synchronisation or Cancel to cancel.

    Reset Output

    Resets a latched output.

    Events

    Displays the Events menu, so that you can view a list of recent controller events.

  2. When you’re ready, tap ← to return to the Connect to Controller menu.
