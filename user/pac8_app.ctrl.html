Controllers / Doors

This menu displays a list of connected controllers. The top line shows the number of configured controllers.

To configure an existing controller:

  1. Tap the name of the controller.

    The Controller / Door Detail menu is displayed.

  2. When you’re ready, tap ← to return to the Main menu.

Note

You must use the PAC8 Backend web site to add controllers to the system.
