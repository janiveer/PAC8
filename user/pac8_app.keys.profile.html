Key Access Details

This menu enables you to specify a time profile for the key and the areas that the key can access.

To specify key access details:

  1. Enter the following options:

    Time profile

    Select an option from the pop-up menu:

    • Time Profile Name — To apply this time profile.

    • None — To apply no time profile.

    Access

    A list of areas. Toggle each area on or off to specify whether the key can access this area.

  2. Tap Save to save the options or Cancel to cancel.
