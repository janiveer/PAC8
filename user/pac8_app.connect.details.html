Event Details

This menu enables you to see the details of an event.

To view event details:

  1. Check the following options:

    Time

    Read only. The date and time of the event.

    Description

    Read only. A description of the event.

    Keycode

    Read only. The keycode associated with this event. If none, the keycode is listed as all 0s.

  2. For Unknown ID errors only: to add the unknown key to the database, tap Add Key to Database.

    The Key Details menu is displayed.

  3. When you’re ready, tap ← to return to the Events menu.
