Global Settings

To specify the global settings:

  1. Enter the following options:

    Site Name

    The location of the PAC8 controllers.

    System Access PIN

    The PIN (Personal Identification Number) used to unlock the door, if PIN/Prox readers are being used.

    System Access PIN Length

    The number of digits in the System Access PIN.

    System Alarm PIN

    The PIN (Personal Identification Number) used to reset the alarm output, if PIN/Prox readers are being used.

    System Alarm PIN Length

    The number of digits in the System Alarm PIN.

    Daylight Saving

    Toggle on or off. When this option is switched on, controllers adjust for daylight saving time automatically, using European daylight saving rules (2000/84/EC).

  2. Tap Save to save the options or Cancel to cancel.
